  3. Inicialização/desligamento e Login do usuário

    1. Instalação do HDD (ignore isto se você não precisar de armazenamento):

      Instale o HDD (ignore isto se você não precisar de armazenamento):

    2. Inicializar

      Conecte à energia, o indicador acende, o DVR é inicializado.

      1. Use o adaptador correspondente para evitar situações anormais ou danos.
      2. Desligue o interruptor de energia do aparelho (alguns podem ter interruptores)

    3. Para desligar o dispositivo, existem Desligamento-suave e Desligamento-forçado.

      Desligamento-suave: Vá em MainMenu Menu Principal- Logout Sair- Shutdown Desligar

      Desligamento-forçado: Conecte a energia no painel traseiro.

      1. Recuperação de energia: Quando o DVR é desligado de forma anormal, reinicie-o, todas as gravações serão salvas automaticamente e restauradas para o status anterior.
      2. Substituir o HDD: Desligue a energia antes de substituir.

    4. Login do Sistema

      Quando o DVR é inicializado, o usuário deve efetuar login e o sistema fornece a função correspondente com a permissão do usuário. Existem duas configurações de usuário. Os nomes são admin, padrão. Admin é o padrão sem proteção por senha: Se a senha estiver continuamente errada três vezes, o alarme será iniciado. Se a senha estiver errada por cinco vezes, a conta será bloqueada. (Ao reiniciar ou esperar meia hora, a conta será desbloqueada automaticamente).

  4. Configurações de rede

    Vá em Main Menu Menu Principal > System Sistema > Network Rede, do seguinte modo:

    IP Adress Endereço de IP: o padrão é 192.168.1.10;

    Subnet Mask Máscara de sub-rede: o padrão é 255.255.255.0;

    Gateway IP de Entrada: o padrão é 192.168.1.1

    O endereço IP da cúpula de velocidade deve compartilhar o mesmo segmento de rede com o roteador e estar de acordo com seu IP, em vez de entrar em conflito com outro IP. Para garantir o acesso do dispositivo à rede vá em main menu menu principal> system sistema> network rede, selecione DHCP Enable Habilitar DHCP (o roteador deve habilitar o DHCP).

  5. Configurações de Gravação

    O DVR deve conter HDD da seguinte forma:

    1. Gerenciamento de HDD:

      Depois de instalar o HDD, inicialize o DVR, vá em main menu menu principal- Advanced Avançado- Storage Armazenamento para formatar o HDD. Em seguida, defina o HDD como read-write leitura/gravação. Pelo menos um disco de leitura/gravação é necessário para a gravação normal.

    2. Configurações de vídeo

      Vá em main menu menu principal- record gravação- record conf. configurações de gravação

      Channel Canal Escolha o número do canal correspondente. Escolha a opção all todos para definir os canais inteiros.

      Redundancy Redundância Escolha a função de redundância para duplicar o backup(salvamento) do arquivo.

      Length Comprimento Defina o tempo de duração de cada arquivo de vídeo entre 1min a 120min. 60 minutos é o valor padrão.

      Mode Modo Defina o estado do vídeo: schedule agendar, manual ou stop parar.

      Configuração: Gravação de acordo com o tipo de vídeo definido (common comum, detection detecção) e seção de tempo.

      Manual: O canal correspondente começa a gravar 24 horas sem parar.

      Stop: Independentemente do estado do canal, a gravação do canal correspondente é interrompida quando o botão Stop é selecionado.

      Period Período Definir a seção de tempo da gravação comum, a gravação começará apenas entre o intervalo definido.

      Record type Tipo de gravação Definir tipo de gravação: regular, detection detecção, alarm alarme

      Regular: Grava de acordo com a seção de tempo definido. O tipo de arquivo de vídeo é "R".

      Detect: Dentro da seção de tempo definido, acionar a detecção de movimento, vídeo cego, perda de vídeo ou análise anormal. Quando o alarme acima está definido como opening recording gravação de abertura, o estado de detecção de gravação está ativado. O tipo de arquivo de vídeo é M.

      Alarme: Acione o sinal de alarme externo na seção de tempo definido. Quando o alarme acima é definido como opening recording gravação de abertura, o estado "detection recording gravação de detecção" está ativado. O tipo de arquivo de vídeo é "A".
